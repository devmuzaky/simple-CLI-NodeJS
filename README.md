# simple-CLI-NodeJS
Simple CLI to add or update or delete USERS in JSON File using NodeJS

in this CLI  we can create user with : 

    1. unique id [i create it using Data.now()]
    2. name
    3. age


#Usage: users-management [options] [command]

#CLI to manage users data

Options:

 output the version number:
 
        -V, --version
  
display help for command

        -h, --help       

Commands:

  add [options]     =>  add new user
  
  list              =>  display all users
  
  delete <string>   =>  delete a user by ID
  
  help [command]    =>  display help for command
